Process

Bookkeepr started as a learning project. The idea was to create a space where anybody can list books they love, list books they want to read, catagorize volumes, etc. It was quick, simple, and fun.

Technology

Bookkeepr's backend is written using Flask, Jinja2, and PostgreSQL. The frontend uses Bootstrap with scripting components written using JavaScript.
